    Basic Swagger Combine Example v1.0.0

    Scroll down for code samples, example requests and responses. Select a language for code samples from the tabs above or the mobile navigation menu.

    An API for the pilot sharing economy project DELING. The DELING API is organized around REST. Our API has resource-oriented URLs and uses HTTP response codes to indicate API errors. The resources are different domains such as Car, Boat, Housing and Services (cleaning, baby-sitting etc). We support CORS, allowing you to interact securily with our API from a client-side web application. You should never expose your secet API key in any public website's client-side code. JSON is returned by all API responses, including errors.

    Authentication

    • API Key (access_token)
      • Parameter Name: Authorization, in: header. Copy 'Bearer ' + valid acccess token into field

    • oAuth2 authentication.

      • Flow: implicit
      • Authorization URL = http://petstore.swagger.io/oauth/dialog
    Scope Scope Description
    write:pets modify pets in your account
    read:pets read your pets
    • API Key (api_key)
      • Parameter Name: api_key, in: header.
